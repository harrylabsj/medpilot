# MedPilot Release Readiness Checklist

## Done in this skeleton
- project scaffold created
- TypeScript build configured
- CLI entrypoints available
- MVP data models defined
- initial rules and tests added
- HealthManual base structure supported

## Before any real release
- add persistent storage
- add API or app surface
- integrate reminder scheduler
- add OCR and transcript ingestion
- add more medical safety rules with review
- add audit logs and user separation
- define deployment target
- define privacy and data retention policy
- add CI pipeline and release artifact generation

## Current recommendation
Treat this repo as a development skeleton only, not a production release candidate.
